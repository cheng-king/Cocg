# 🤖 Cocg

> AI 辅助搭建的内部知识库 Agent 项目

Cocg 是一个基于 AI 技术的智能知识库助手，旨在通过多 Agent 协作模式，高效处理企业内部文档，提供精准的知识检索与问答服务。

## 🚀 核心功能

- **多格式支持**：支持上传 PDF、Word、Markdown 等多种格式文档。
- **智能解析**：自动对文档进行切片、向量化处理。
- **多 Agent 协作**：集成 Router Agent、Reasoning Agent 等，实现复杂任务的自动化处理。
- **流畅交互**：基于 Framer Motion 构建的丝滑动画体验。

## 🛠️ 技术栈

- **框架**：React
- **动画**：Framer Motion
- **图标**：Lucide React
- **样式**：Tailwind CSS (如果用了的话)
